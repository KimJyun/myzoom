# 온라인 화상 채팅 프로그램

## Installation

> Nodemon 설치<br/>
    ```
    npm i nodemon -D
    ```
    
> Bable 설치<br/>
    ```
    npm i @babel/core @babel/cli @babel/node @babel/preset-env -D
    ```
    
> express 설치<br/>
    ```
    npm i express
    ```
    
> pug 설치<br/>
    ```
    npm i pug
    ```

> ws library 설치<br/>
    ```
    npm i ws
    ```

> socket.io 설치<br/>
    ```
    npm i nodemon -D
    ```
    
> 실행<br/>
    ```
    npm run dev
    ```
