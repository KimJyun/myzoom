## Myzoom

Node.js, socket.io를 사용하여 나만의 Zoom 만들어보기  
Create your own Zoom using Node.js.

## Configuration

1. Frontend: pug
2. Backend : Node.js
3. A W S : EC2(Ubuntu)

## Quickstart

### Prerequisite
- node v14.x
- npm 6.x

## Installation

> 쉬운 설치<br/>
    ```
    npm i
    ```
> 사용 모듈별 설치

>> Nodemon 설치<br/>
    ```
    npm i nodemon -D
    ```
    
>> Bable 설치<br/>
    ```
    npm i @babel/core @babel/cli @babel/node @babel/preset-env -D
    ```
    
>> express 설치<br/>
    ```
    npm i express
    ```
    
>> pug 설치<br/>
    ```
    npm i pug
    ```

>> ws library 설치<br/>
    ```
    npm i ws
    ```

>> socket.io 설치<br/>
    ```
    npm i socket.io
    ```
    
>> socket.io-stream 설치<br/>
    ```
    npm i socket.io-stream
    ```
    
> 실행<br/>
    ```
    npm run dev
    ```
