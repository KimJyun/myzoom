# 온라인 화상 채팅 프로그램

## Installation

> Nodemon 설치
    ```
    npm i nodemon -D
    ```
    
> Bable 설치
    ```
    npm i @babel/core @babel/cli @babel/node @babel/preset-env -D
    ```
    
> xpress 설치
    ```
    npm i express
    ```
    
> pug 설치
    ```
    npm i pug
    ```

> ws library 설치
    ```
    npm i ws
    ```

> socket.io 설치
    ```
    npm i socket.io


> 실행
    ```
    npm run dev
    ```
